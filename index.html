Nitrogen Dioxide (NO2) and Nurseries

This map shows London nureseries relative to NO2 concentrations in the air, from background sources and at the roadside.

The EU limit value for mean concentration of NO2 is 40 µg/m3.

For a full list of data sources and a description of the method used to derive distances see the github page.
